# Contributing

This repository is primarily a personal learning course and portfolio, but contributions can still follow a clear standard.

## Contribution Types

- Fix typos or broken links.
- Improve explanations.
- Add exercises.
- Add tests or eval cases to projects.
- Add references to durable primary sources.

## Content Standard

Course content should be:

- Original
- Practical
- Clear for self-study
- Respectful of source copyrights
- Linked to references when inspired by external material

Do not paste paid course transcripts, proprietary slides, or long copied passages from books or websites.

## Chapter Standard

Each chapter should include:

- Overview
- Theory
- Exercises
- Labs
- Solutions
- Interview questions
- References
