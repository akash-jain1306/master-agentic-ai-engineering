## Course Benchmark

The paid Udemy outline from the planning chat is useful as a benchmark for pacing and breadth:

- Week 1: foundations, setup, first API call, first agent loop
- Week 2: OpenAI Agents SDK, structured outputs, tools, guardrails, deep research
- Week 3: CrewAI and multi-agent crews
- Week 4: LangChain, LangGraph, long-running agents, human review
- Week 5: Google ADK, Microsoft, AWS, Pydantic AI, Agno, Mastra, multi-framework comparison
- Week 6: MCP, harness engineering, custom servers, capstone, deployment

This repository should use that only as a planning reference. The explanations, labs, exercises, and projects should be original.
