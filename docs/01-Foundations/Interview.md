# Interview Preparation

## Beginner

1. What is an LLM?
2. What does an AI engineer do?
3. Why can LLMs produce different answers to the same prompt?
4. What is a hallucination?
5. What is the purpose of an evaluation set?

## Intermediate

1. How would you test an LLM-powered summarization feature?
2. What should be logged when an AI system calls a tool?
3. How do you decide whether to solve a task with rules, retrieval, or an LLM?
4. What are common risks when user input is inserted into a prompt?
5. What makes an agentic workflow harder to debug than a single model call?

## Senior

1. Design the rollout plan for a high-volume AI support assistant.
2. How would you build an eval harness for a tool-using agent?
3. What production metrics would you monitor after launch?
4. How would you contain the blast radius of a bad model output?
5. Explain how you would balance model quality, latency, cost, and safety.
