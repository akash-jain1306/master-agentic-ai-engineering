# 01 - Foundations

This chapter establishes the engineering baseline for the rest of the course. Before building agents, retrieval systems, or production AI services, you need a clear mental model of what AI engineering is and how it differs from ordinary application development.

## Learning Objectives

By the end of this chapter, you should be able to:

- Explain the role of an AI engineer in a product team.
- Distinguish deterministic software behavior from probabilistic model behavior.
- Describe the basic lifecycle of an AI feature.
- Identify common failure modes in LLM-powered systems.
- Set up a clean repository structure for labs and projects.
