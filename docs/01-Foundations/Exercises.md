# Exercises

## Concept Checks

1. In your own words, define AI engineering.
2. Why are LLM-powered systems harder to test than deterministic functions?
3. What is the difference between a chatbot and an agentic system?
4. Name five production concerns that exist outside the model itself.
5. Why should prompts and evals be treated as production assets?

## Applied Thinking

1. Choose one AI feature you use often. Break it into model, interface, data, tools, evals, and monitoring.
2. Design three failure cases for an AI email assistant.
3. List the logs you would want for a customer-support summarization feature.
4. Write a one-paragraph risk assessment for an AI coding assistant.
5. Describe when you would prefer a deterministic rule over an LLM call.

## Design Drill

Design a minimal AI study coach. Include:

- User input
- Model instruction
- State that must be stored
- Tools the system may use
- Stop condition
- Three evaluation examples
