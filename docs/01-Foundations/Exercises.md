# Exercises

Complete these before reading `Solutions.md`.

## Concept Checks

1. Why did ChatGPT create a platform shift when earlier AI systems did not?
2. Explain the difference between an LLM application, a copilot, and an AI agent.
3. Why is an LLM not enough to build a production AI application?
4. What does it mean to say that context is working memory?
5. What problem do tools solve for an LLM?
6. What problem does memory solve?
7. What problem does an evaluation harness solve?
8. Why are most production agents only partially autonomous?

## Applied Thinking

1. Pick three AI products you use, such as ChatGPT, Claude, Cursor, GitHub Copilot, Perplexity, or Gemini. For each one, identify likely examples of context, memory, tools, and evaluation.
2. Choose one workflow from your job or studies. Decide whether it should be a deterministic app, an LLM app, a copilot, or an agent. Explain why.
3. Identify one task where full autonomy would be unsafe. Add two human approval gates.
4. Choose one framework from the ecosystem map. Explain which layer it belongs to and what problem it solves.
5. Write a one-page explanation of how an AI coding assistant answers a debugging question.
