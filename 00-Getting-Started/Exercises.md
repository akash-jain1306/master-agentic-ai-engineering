# Exercises

## Setup Exercises

1. Install Git and Python.
2. Create a GitHub account if you do not already have one.
3. Clone or open this repository locally.
4. Create a virtual environment.
5. Create a private `.env` file with placeholder API key names.

## Reflection Exercises

1. Write a one-page explanation of why you want to learn agentic AI engineering.
2. List three AI products you admire and identify their likely AI engineering components.
3. List three risks you want to understand before building autonomous agents.
4. Define what a successful capstone project would look like for you.
