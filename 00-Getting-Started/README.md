# 00 - Getting Started

This module helps you prepare your environment, understand the course workflow, and begin learning in a structured way.

## Goals

- Set up the local development environment.
- Understand how the repository is organized.
- Learn how to study each chapter.
- Prepare API keys and local tools safely.
- Create the first learning journal entry.

## Files

- `Setup.md`: Local tools, Python environment, API keys, and Git workflow
- `Roadmap.md`: Suggested study path
- `Resources.md`: First resources to watch or read
- `Exercises.md`: Setup and reflection exercises
